開発用に利用する SharePoint 2013 を入手する方法はいくつかありますが、ここでは各種サブスクリプションでの提供状況をご紹介します。

これから SharePoint 開発でビジネスしようと本気で考えている方、[Visual Studio Professional 2012 with MSDN](http://www.microsoft.com/visualstudio/jpn/products/visual-studio-professional-2012#product-edition-professional) を購入して Visual Studio も SharePoint も一緒に入手するのがおすすめですよ。(高いですが・・・)

もし会社を立ち上げたばかりということであれば、[BizSpark](http://www.microsoft.com/ja-jp/mic/bizspark/default.aspx) が超おすすめです。
最大３年間無償で Visual Studio も SharePoint も使いたい放題です。

学生さんや教員の方々は [DreamSpark](http://msdn.microsoft.com/ja-jp/hh455217.aspx) ですね。
SharePoint 2013 を入手可能なサブスクリプション一覧

- BizSpark
- BizSpark Admin
- DreamSpark Premium
- MSDN Platforms
- Visual Studio Professional 2012 with MSDN
- Visual Studio Premium 2012 with MSDN
- Visual Studio Ultimate 2012 with MSDN

無償じゃないと、ということであれば、以下の SharePoint Foundation か評価版をダウンロードしてお使いください。

- [SharePoint Foundation 2013](http://www.microsoft.com/ja-jp/download/details.aspx?id=35488)
- [SharePoint Server 2013 評価版](http://technet.microsoft.com/ja-jp/evalcenter/hh973397.aspx)

なお、これらの情報は記事執筆時点のものとなりますので、ご了承ください。
